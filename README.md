# Ask the expert

- Open your terminal and clone this repository

    First make sure you have a ssh key if you dont have go to my dotfiles repository and follow the instructions

    ```
    git clone git@github.com:ayushkpai/ask-the-expert.git
    ```

- Next install python 

    Also documented in dotfiles

- Create .env

  ```
  touch .env
  ```

- Enter path

  Create a capital.txt

  ```
  touch capital.txt
  pwd
  echo <what pwd returned/capital.txt> >> .env
  ```
